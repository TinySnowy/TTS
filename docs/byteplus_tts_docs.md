

You can interact with the API through HTTP or Websocket requests from any language, or via our demos (Python, Java & Go).

## Text-to-Speech Features
<br>

| Feature | Description |
| :-- | :-- |
| API type | Streaming/non-streaming API |
| Number of voices | 17 pre-trained voices |
| Algorithm effect | Near-human expressions in naturalness, sound quality, rhythm, breath control, emotion, and modal particles |
| Supported languages | Chinese, English, Japanese, and Spanish |
| Speech Synthesis Markup Language (SSML) | SSML is supported. For more information, see SSML doc |
| Integration | See streaming & non-streaming TTS API doc. |
| Deployment | Public cloud |
| Sample rate | 24k (16k and 8k to be supported) |
| Audio format | PCM (default), OPUS, and MP3 |\
|| Note: The WAV format does not support streaming. |
| Speed adjustment | Speed can be adjusted |
